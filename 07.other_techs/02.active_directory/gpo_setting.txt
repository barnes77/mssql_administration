	1. GPO to set up local admins
		• Path: Computer Configuration >> Policies >> Windows Settings >> Security Settings >> Restricted Groups
		• Settings:
			Right click: Add Group.. >> Provide name >> This group is a member of: Administrators
	2. GPO to set up time in domain:
		a. Primary DC
			§ Path: Computer Configuration >> Policies >> Administrative Templates >>  System >> Windows Time Service
			§ Settings:
				□ Enable Windows NTP Server: Enabled
			§ Link: OU of Domain Controllers
			§ Security Filtering: CO of PDC
			§ Additionally configure Time Servers on the PDC by running commands from elevated command line:
				tzutil /s "Central European Standard Time"
				net stop w32time
				w32tm /unregister
				w32tm /register
				net start w32time
				w32tm /config /syncfromflags:manual /manualpeerlist:"vega.cbk.poznan.pl ntp.itl.waw.pl ntp.elpromaelectronics.com zegar.umk.pl" /reliable:yes /update
				w32tm /resync /rediscover /nowait
				w32tm /config /reliable:yes
		b. Other computers 
			§ Path: Computer Configuration >> Policies >> Administrative Templates >>  System >> Windows Time Service
			§ Settings: 
				□ Configure Windows NTP Client
					® NtpServer: FQDN of PDC
					® Type: NTP
					® SpecialPollInterval: 64
				□ Enable Windows NTP Client: Enabled
			§ Path: Computer Configuration >> Policies >> Windows Settings >> Security Settings >> System Services
			§ Settings:
				□ Windows Time: Define this policy setting >> Select service startup mode: Automatic
			§ Additionally in case of any issues run these commands from elevated command line:
				net stop w32time
				w32tm /unregister
				w32tm /register
				gpupdate /force
				net start w32time
				w32tm /config /syncfromflags:domhier /update
				net stop w32time
				net start w32time
				w32tm /resync /rediscover /nowait
	3. GPO to set up time zone:
		• Path: Computer Configuration >> Preferences >> Windows Settings >> Registry
		Click: New >> Registry Wizard
		Select: Local (or computer that is source)
		Grab all settings from \HKLM\System\CurrentControlSet\Control\TimeZoneInformation\
	4. GPO to sign off users:
		• Path: Computer Configuration >> Policies >> Administrative Templates >> Windows Components >> Remote Desktop Services >> Remote Desktop Session Host >> Session Time Limits
		• Settings:
			§ Set time limit for active but idle Remote Desktop Services sessions
			§ Set time limit for active Remote Desktop Services sessions
			§ Set time limit for disconnected sessions: 3 days
	5. GPO for password policies:
		• Path: Computer Configuration >> Policies >> Windows Settings >> Security Settings >> Account Policies >> Password Policy
		• Settings:
			§ Enforce password history
			§ Maximum password age
			§ Minimum password age
			§ Minimum password length
			§ Password must meet complexity requirements
			§ Store passwords using reversible encryption
		• Path: Computer Configuration >> Policies >> Windows Settings >> Security Settings >> Account Policies >> Account Lockout Policy
		• Settings:
			§ Account lockout duration
			§ Account lockout threshold
			§ Reset account lockout counter after
		• Additional info about default complexity:
			Not contain the user's account name or parts of the user's full name that exceed two consecutive characters
			Be at least six characters in length
			Contain characters from three of the following four categories:
			English uppercase characters (A through Z)
			English lowercase characters (a through z)
			Base 10 digits (0 through 9)
			Non-alphabetic characters (for example, !, $, #, %)
			Complexity requirements are enforced when passwords are changed or created.
	6. GPO to disable UAC:
		• Path: Computer Configuration >> Policies >> Windows Settings >> Security Settings >> Local Policies >> Security Options
		• Settings:
			§ User Account Control: Admin Approval Mode for the Built-in Administrator account: Enabled 
			§ User Account Control: Behavior of the elevation prompt for standard users: Automatically deny elevation requests 
			§ User Account Control: Detect application installations and prompt for elevation: Enabled 
			§ User Account Control: Only elevate UIAccess applications that are installed in secure locations: Enabled 
			§ User Account Control: Run all administrators in Admin Approval Mode: Enabled 
			§ User Account Control: Switch to the secure desktop when prompting for elevation: Enabled 
			§ User Account Control: Virtualize file and registry write failures to per-user locations: Enabled 
	7. GPO to set up high performance power plan:
		• Path: Computer Configuration >> Policies >> Administrative Templates >> System 
		• Setting:
			§ Select Display Shutdown Event Tracker: Disabled
	8. GPO to disable shutdown event tracker
		• Path: Computer Configuration >> Policies >> Administrative Templates >> System >> Power Management
		• Setting:
			§ Select an active power plan: Automatic (recommended)
	9. GPO for wallpaper:
		• Path: User Configuration >> Policies >> Administrative Templates >> Desktop >> Desktop
		• Settings:
			§ Desktop Wallpaper: Enabled // UNC path to wallpaper in .jpg
	10. GPO for screensaver:
		• Path: User Configuration >> Policies >> Administrative Templates >> Control Panel >> Personalization
		• Settings:
			§ Enable screen saver: Enabled
			§ Password protect the screen saver: Enabled
			§ Screen saver timeout: Enabled // n seconds
			§ Force specific screen saver: Enabled // path to .scr file (e.g. %Systemroot%\System32\example.scr)
