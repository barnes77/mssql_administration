**CONTENT**

Scripts published include cmd commands, T-SQL queries obviously and PowerShell queries. The language used should be easy to spot, besides the script itself, in name file and its extension. Please note that I've done my best to clearly state whether the query does perform a change in anything or is merely a read-only operation, but if you have doubts and do not understand what the script does, please refrain from running it especially against Production servers.

The repository consists of more than 300 files and is a result of many years of my work and exchanging experience with colleagues in a few companies. Please note that despite my efforts please expect that:

* some of the scripts do not use the possibilities of the newest SQL Servers, e.g. the newest DMVs

* some of the scripts are written for newer versions of SQL Server and are not backwards compatible with older versions (pre-SQL2012)

* some of the scripts might not be the most elegant way of solving things / gathering information, but they might have been written when I started as a DBA and there has been no need to update them ever since
